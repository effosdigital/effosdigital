🚀 Publicar alterações para todo mundo

Tudo que você edita aqui fica salvo só neste navegador — outros visitantes não veem essas mudanças. Para publicar de vez: edite tudo normalmente abaixo, clique em "Baixar data.js atualizado", e depois suba o arquivo baixado substituindo assets/js/data.js no seu repositório do GitHub (pode subir direto pela interface web do GitHub, arrastando o arquivo). Assim que o GitHub Pages atualizar, a mudança vale para qualquer pessoa que acessar o site.

🔐 Conta e recuperação de senha

Este site é 100% estático (sem servidor próprio), então o envio real de e-mail de recuperação depende do EmailJS (plano gratuito disponível). Crie uma conta lá, configure um serviço de e-mail e um template com a variável {{code}}, e cole as chaves abaixo. Sem essa configuração, o código de recuperação aparece na tela em "modo de teste" para você não ficar sem acesso.
